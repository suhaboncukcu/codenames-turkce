# Codenames Türkçe

A simple, static web implementation of the popular board game **Codenames** in Turkish.

## Features

-   **Single Player / Local Multiplayer**: Play on a single device by passing it around or projecting the screen.
-   **Spymaster View**: Toggle to see the colors of all cards.
-   **Responsive Design**: Works on mobile and desktop.
-   **Turkish Word List**: Includes a curated list of Turkish words.

## How to Play

1.  **Teams**: Divide into two teams: Red and Blue. Each team chooses a Spymaster.
2.  **Objective**: Spymasters give one-word clues to help their team guess their words while avoiding the other team's words and the Assassin (Black card).
3.  **Winning**: The first team to find all their agents wins. If a team picks the Assassin, they lose immediately.

## Development

This project uses standard HTML, CSS, and JavaScript with no build steps required.

### Structure
-   `index.html`: Main entry point.
-   `style.css`: All styles.
-   `script.js`: Game logic.
